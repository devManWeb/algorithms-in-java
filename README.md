# algorithms-in-java
Collection of search, sort and ADT algorithms, written in Java 

* Dynamic array
* Linked list
* Linear search - O(n^2)
* Binary Search - O(logn)
* Insertion sort + Inverse Insertion Sort - O(n^2), but O(n) when inserting on an already sorted array
* Selection Sort - O(n^2)
* Merge sort - O(n logn)
* Bubble sort - O(n^2)
